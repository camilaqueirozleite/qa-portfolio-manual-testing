Test Plan - SauceDemo
1. Introduction
This document describes the testing approach for the SauceDemo web application.

The goal is to ensure that main functionalities work correctly and provide a good user experience.

2. Scope
In Scope:
Login functionality
Product listing page
Add to cart feature
Checkout process
Out of Scope:
Backend database testing
Performance testing
Security testing
3. Test Approach
Manual testing will be performed by executing test cases step-by-step and validating expected results.

We will use:

Exploratory testing
Functional testing
Negative testing
4. Test Environment
Browser: Google Chrome
Operating System: Windows / Mac
Website: https://www.saucedemo.com
5. Test Data
Valid user credentials (provided by SauceDemo)
Invalid login attempts
Product selection scenarios
6. Risks
Demo website may reset data
Limited backend visibility
